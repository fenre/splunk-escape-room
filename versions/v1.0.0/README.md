# v1.0.0 — First Major Release (25 Mar 2026)

Snapshot of the v1.0.0 release for reference. The live codebase has moved to v2.0.0.

## What's in v1.0.0

- **7 linear vault seals** (no act structure)
- **3 indexes**: nakatomi_access, nakatomi_vault, nakatomi_building
- **7 sourcetypes**, **3 lookup tables**
- **~830 events** generated by `generate.py`
- **6 Splunk dashboards**: Mission Brief, Search Terminal, Access Terminal, Vault Terminal, Building Systems, Terminal
- **game.html**: Dual-mode (physical/digital), 7-seal code entry, 7 styled clue cards, ROT13 decoder, power element (Seal 7), 60-minute timer, 3-error limit

## Key differences from v2.0

| Feature | v1.0.0 | v2.0.0 |
|---|---|---|
| Structure | 7 linear seals | 5 acts, 26 tasks |
| Events | ~830 | ~4,000 |
| Indexes | 3 | 4 (+nakatomi_comms) |
| Sourcetypes | 7 | 12 |
| Lookups | 3 | 7 |
| Timer | 60 min | 90 min |
| Error limit | 3 | 7 |
| Hints | None | Progressive (4 levels) |
| Scoring | None | Points system |
| Dossier | None | Intel collection panel |
